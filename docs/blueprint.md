# **App Name**: PolyJournal

## Core Features:

- Instant Wallet Connect: Seamlessly connect to Polymarket via RainbowKit and WalletConnect v2.
- Real-time Trade Import: Automatically import Polymarket positions and closed trades using the official Gamma API and clob.polymarket.com/positions endpoint.
- Gorgeous Trade Timeline: Visualize trades with market questions, entry price, position size, PnL, color-coded outcomes, and editable thesis fields.
- Local-First Storage with Sync: Securely store data using IndexedDB with automatic iCloud/Chrome sync.
- One-Click Export: Export trade data in CSV & PDF formats for easy tax reporting.
- Stats Dashboard: Display key metrics such as total PnL curve, win rate by category, average hold duration, and conviction score.
- AI Thesis Reviewer: Provide feedback on trading theses using Claude-3.5-Sonnet or Grok-4 as a tool to provide suggestions and potential improvements.

## Style Guidelines:

- Primary color: Vibrant purple (#A020F0) to represent Polymarket's forward-thinking approach.
- Background color: Dark gray (#282828) for a sleek and modern look.
- Accent color: Electric blue (#7DF9FF) to highlight key information and interactive elements.
- Body font: 'Inter', a sans-serif for clean and readable text.
- Headline font: 'Space Grotesk', a sans-serif for headlines and titles; pairs well with 'Inter'.
- Smooth, buttery animations using Framer Motion or CSS to provide a premium user experience.
- Mobile-first PWA design, ensuring a seamless experience across all devices with installable home screen access.
